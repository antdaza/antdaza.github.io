ANTD Coin Specifications

ANTD Coin is engineered for privacy, decentralization, and secure communication. Below are the current protocol-level specifications.

Consensus

  • Proof of Work: RandomX
  • Block Time: 2 minutes (same as Monero)
  • Difficulty Algorithm: Dynamic (Monero-compatible)

Block Rewards

  • Total Reward per Block: 600 ANTD
  • Miner Reward: 200 ANTD
  • Full Node Reward: 300 ANTD
  • Community Development Fund: 100 ANTD

Network Architecture

  • Transaction Privacy: RingCT, stealth addresses
  • Message/Article Support: On-chain via tx_extra
  • Community Access: Articles viewable via JSON RPC

Supply & Inflation

  • Emission: Tail emission compatible with Monero
  • Inflation Model: Logarithmic decrease toward flat subsidy
  • Block Emission Tail: Minimum ongoing rewards for miners and nodes
