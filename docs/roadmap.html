ANTD Development Roadmap

Here’s what’s ahead for ANTD Coin as we build a powerful, private communication + transaction network.

Phase 1 – Launch (✓ Completed)

  • Monero codebase fork with full privacy features (RingCT, stealth addresses)
  • Basic article/message posting functionality using `tx_extra`
  • CLI interface for `add_article` command
  • Basic block explorer with article parsing

Phase 2 – Community Messaging

  • Web wallet integration for message posting
  • Frontend interface to view community messages
  • Rate-limiting and anti-spam rules
  • Allow third-party publishers to post on behalf of users

Phase 3 – Node Rewards & Expansion

  • Implement full-node tracking and uptime measurement
  • Introduce staking or uptime-based rewards for running a full node
  • Incentives for hosting articles and messages

Phase 4 – Smart Communication Layer

  • Support for longer messages (beyond 100 words) with segmentation
  • Enable message signing, expiration, and edit logic (soft-updates)
  • Optional encrypted metadata (author identity, category)
  • Public message feeds and channels
